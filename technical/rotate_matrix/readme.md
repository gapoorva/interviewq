# Rotate Matrix

Given an NxN matrix of numbers, rotate the matrix by 90 degrees clockwise. Do this in place, meaning use constant space.

Example:
```
[ 0 1 2 ]
[ 3 4 5 ]
[ 6 7 8 ]
```
becomes:
```
[ 6 3 0 ]
[ 7 4 1 ]
[ 8 5 2 ]
```
